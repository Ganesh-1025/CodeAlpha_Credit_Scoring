# CodeAlpha - Credit Scoring Prediction

## Project Overview

This project predicts whether a loan applicant is creditworthy using Machine Learning.

## Dataset

* Credit Risk Dataset

## Algorithms Used

* Logistic Regression
* Random Forest Classifier

## Libraries

* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## Workflow

1. Data Loading
2. Data Preprocessing
3. Missing Value Handling
4. Encoding Categorical Features
5. Train-Test Split
6. Feature Scaling
7. Logistic Regression
8. Random Forest
9. Model Evaluation
10. Feature Importance Analysis

## Results

Random Forest achieved better performance than Logistic Regression on this dataset.
